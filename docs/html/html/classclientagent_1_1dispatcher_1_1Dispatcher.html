Detailed Description

This class provides an abstract interface into the dispatcher itself.

It is intended to centralize all dispatcher-related calls and interfacing into a single location such that any changes only affect the one location.

Under Linux, this will wrap the dispatcher scripts and perform the same function as the DispatcherHelper class in the original Linux Client Agent.

Under Windows, this will simply handle the platform-specific calls and scripts that are relevant.
